# Usage

To use MkDocs Slides in your MkDocs project, follow these steps:

1. **Add the plugin to your `mkdocs.yml` configuration file:**

   ```yaml
   plugins:
     - slides
   ```

2. **Create a slide decks using Markdown files.** like this :

```text
```slides
title: My Presentation
url_stub: my-pres
nav:
    - slides/presentation/*.md```
```
