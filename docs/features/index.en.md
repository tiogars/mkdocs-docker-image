# Features

- python environment [3.14.0-slim](https://hub.docker.com/_/python/)
- static documentation generation with [MkDocs](https://www.mkdocs.org/)
- "Material for MkDocs" theme
    - [Website](https://squidfunk.github.io/mkdocs-material/)
    - [Github](https://github.com/squidfunk/mkdocs-material)
- multilingual support with "mkdocs-static-i18n"
    - [Website](https://ultrabug.github.io/mkdocs-static-i18n/)
    - [Github](https://github.com/ultrabug/mkdocs-static-i18n)
- export PDF of the entire site with "mkdocs-to-pdf":
    - [Website](https://mkdocs-to-pdf.readthedocs.io/en/latest/)
    - [Github](https://github.com/domWalters/mkdocs-to-pdf)
    - [Pypi](https://pypi.org/project/mkdocs-to-pdf/)
- search with "search"
- diagrams with "mkdocs-kroki-plugin"
    - [Website](https://kroki.io/)
    - [Github](https://github.com/fralau/mkdocs-kroki-plugin)
- printing a specific page with "mkdocs-print-site-plugin"
    - [Github](https://github.com/timvink/mkdocs-print-site-plugin)
